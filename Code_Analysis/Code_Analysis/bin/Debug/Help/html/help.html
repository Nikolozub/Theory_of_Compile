Описание программы

Программа Анализатор кода является текстовым редактором. Это значит, что данная программа предназначена для работы с текстом. В ней мы можем вводить текст и редактировать его. Давайте запустим АК. Для этого щелкаем мышкой по значку программы. Откроется окно программы.

Описании функций

Создать  

Создает новый документ

Открыть  

Открывает документ с расширением *.txt

Сохранить  

Сохраняет документ с расширением *.txt

Созранить как  

Сохраняет документ с измененным названием и расширением *.txt

Вырезать  

Вырезает выделеный текст в буфер обмена

Копировать  

Копирует текст в буфер обмена

Вставить  

Вставляет текст из буфера обмена

Удалить  

Удаляет выделеный текст

Удалить всё  

Удалить весь текст в документе

Отменить  

Отменяет последнее совершенное действие

Повторить  

Возвращает последнюю совершенную отмену

Пуск  

Запускает программу обработки кода

Справка  

Открывает справку по анализатору кода

Выделить всё

Выделяет весь текст в текстовом поле.

Регулярные выражения

Мы ищем номера банковских карт, которые содержат в себе 16 символов и записываются в виде

Вид ввода номера банковской карты:

yxxx xxxx xxxx xxxx

yxxx-xxxx-xxxx-xxxx

yxxxxxxxxxxxxxxx

Где "x" является числом от 0 до 9

Где "y" является числом от 2 до 6

Современный номер банковской карты состоит из 16 цифр. Первая цифра означает платежную систему:

Первая цифра банковской карты Платёжная система
2 Мир
3 American express, Maestro, JCB
4 Visa
5 Mastercard, Maestro
6 Maestro, Union pay

Регулярное выражение:

\b[2-6]([0-9]{15}|[0-9]{3}-[0-9]{4}-[0-9]{4}-[0-9]{4}|[0-9]{3} [0-9]{4} [0-9]{4} [0-9]{4})\b
